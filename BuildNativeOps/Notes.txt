Download libtorch:

https://download.pytorch.org/libtorch/cu121 e.g. for v12.1
https://download.pytorch.org/libtorch/cu118 e.g. for v11.8

Install CUDA Toolkit (12.1):
https://developer.nvidia.com/cuda-12-4-0-download-archive
check nvcc --Version in dev shell

Setting up build env:

Build Tools + Ninja + CMake (+ VSCode)

Ninja:
in dev Shell -> winget install Ninja-build.Ninja

MSVC Compiler with VisualStudio installer
-> Visual Studio Build Tools
  -> Desktop C++
  -> MSVC v143 v14.38 for cuda 12.1
  -> Windows 10 SDK
   

VSCode Extensions:
- CMaker Tools
- CMake
- C++

In VSCode:
Hit Ctrl+Shift+P -> CMake: Configure
Then Ctrl+Shift+P -> CMake: Build

cl.exe to check Compiler Version 

Setting up env variables: call "C:\Program Files (x86)\Microsoft Visual Studio\2022\BuildTools\VC\Auxiliary\Build\vcvars64.bat"


Always start with x64 native Tools command prompt



"C:\Program Files (x86)\Microsoft Visual Studio\2022\BuildTools\Common7\Tools\VsDevCmd.bat" -arch=x64 -host_arch=x64 -vcvars_ver=14.38
